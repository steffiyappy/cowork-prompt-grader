How is this graded?

Cowork credit usage scales on four buckets: Models — the AI model chosen per task (reasoning/output); Runtime — orchestration and long-running work; Context — reading emails, files, meetings and people; Tools — actions taken (building, sending, scheduling). Each bucket is scored 0–3 from the wording of your prompt and combined into a Light / Medium / Heavy grade. This is a heuristic to compare prompts — not a credit prediction. For the real charge, run the task in Cowork and type /cost in the task chat.
